# Compute Nutritional Breakdown

Using `lookupFood()` for each ingredient, compute:

- Per-ingredient calories, protein, carbs, fat, fiber
- Per-serving totals
- Full-meal totals
- Percentage of recommended daily values

Use `scoreMeal()` to generate an overall nutrition score (0-100).

Display the breakdown using `MacroBreakdown` component.

<output target="nutritionBreakdown">
{
  "perServing": { "type": "object" },
  "score": { "type": "number" },
  "flaggedNutrients": { "type": "array" }
}
</output>

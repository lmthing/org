# Gather Meal Details

Collect the meal description and any missing details from the parent agent.

If spawned by a parent agent, use `askParent()` to clarify:
- Number of servings
- Portion sizes (small/medium/large)
- Target calorie range (if any)
- Dietary guidelines to evaluate against

If `askParent()` returns `{ _noParent: true }`, use defaults:
- 2 servings, medium portions, USDA guidelines

Load relevant knowledge from `food-database` to identify ingredients.

<output target="mealDetails">
{
  "description": { "type": "string" },
  "ingredients": { "type": "array" },
  "servings": { "type": "number" },
  "guidelines": { "type": "string" }
}
</output>

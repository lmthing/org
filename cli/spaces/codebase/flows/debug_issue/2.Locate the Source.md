# Locate the Source

Use the diagnostics from reproduction to trace the bug to its source.

## Tracing Strategies

### Frontend Issues
1. Check the component rendering the broken UI
2. Trace state back through hooks (`useFile`, `useAuth`, etc.)
3. Check if the issue is in the app or a shared lib (`@lmthing/ui`, `@lmthing/state`)
4. Verify data coming from cloud functions is correct

### Backend Issues
1. Check the edge function being called
2. Verify request format (headers, body, auth token)
3. Check `_shared/` modules (auth, cors, provider)
4. Verify database queries and RLS policies

### Integration Issues
1. Check the boundary between frontend and backend
2. Verify CORS headers on the edge function
3. Check auth token is being sent correctly
4. Verify SSE streaming if applicable (generate-ai)

### Space/Agent Issues
1. Check agent `config.json` for correct knowledge/component/function references
2. Verify flow step definitions and output schemas
3. Check knowledge option files for correct frontmatter
4. Verify `loadKnowledge` selector matches knowledge structure

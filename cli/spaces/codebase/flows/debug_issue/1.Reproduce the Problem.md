# Reproduce the Problem

Before fixing anything, understand exactly what's broken and reproduce it reliably.

## Steps

1. **Read the bug report** — What's the expected behavior? What's the actual behavior?
2. **Identify the affected area** — Which app, package, or flow is involved?
3. **Reproduce locally** — Follow the reproduction steps in the local dev environment
4. **Gather diagnostics** — Check browser console, network tab, edge function logs, terminal output
5. **Narrow the scope** — Is it a frontend issue, backend issue, or integration issue?

## Diagnostic Tools

- **Browser DevTools** — Console errors, network requests, React DevTools
- **Supabase Dashboard** — Database state, function logs, auth logs
- **Terminal** — Edge function stdout/stderr from `supabase functions serve`
- **nginx logs** — Proxy errors at `/var/log/nginx/error.log`

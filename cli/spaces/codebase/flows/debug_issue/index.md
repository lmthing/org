# Debug Issue

Guides a developer through systematically diagnosing and fixing a bug in the lmthing codebase.

## Steps

1. [Reproduce the Problem](1.Reproduce%20the%20Problem.md) — Understand and reliably reproduce the issue
2. [Locate the Source](2.Locate%20the%20Source.md) — Trace through the system to find where the bug lives
3. [Understand the Context](3.Understand%20the%20Context.md) — Read surrounding code and understand the intended behavior
4. [Apply the Fix](4.Apply%20the%20Fix.md) — Implement a targeted fix following project conventions

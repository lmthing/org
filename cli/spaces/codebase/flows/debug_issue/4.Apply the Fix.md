# Apply the Fix

Implement the fix with minimal blast radius, following project conventions.

## Fix Principles

1. **Minimal change** — fix the bug, don't refactor around it
2. **Same layer** — fix in the layer where the bug lives (don't add workarounds elsewhere)
3. **Convention-consistent** — follow the same patterns as surrounding code
4. **Verify** — reproduce the bug, apply the fix, verify the bug is gone
5. **Regression check** — verify nothing else broke

## Steps

1. Implement the fix in the identified source file
2. If the fix changes shared lib behavior, check all consumers
3. Test the fix against the original reproduction steps
4. Test adjacent functionality for regressions
5. Commit with a clear `fix(scope): description` message

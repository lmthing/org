# Set Up Environment

Verify the developer has all prerequisites installed and configure the local development environment for the lmthing monorepo.

## Prerequisites to Check

- **Node.js** ≥ 20
- **pnpm** ≥ 9
- **Deno** (for cloud/edge functions)
- **Git** configured with GitHub SSH access
- **nginx** (installed by `make proxy`)

## Steps

1. Check each prerequisite version and report any missing tools
2. Clone the repository if not already cloned
3. Run `pnpm install` to install all workspace dependencies
4. Run `make proxy` to set up nginx reverse proxy and `/etc/hosts` entries for `*.test` domains
5. Verify the proxy setup by checking nginx status

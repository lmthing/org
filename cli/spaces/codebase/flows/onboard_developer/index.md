# Onboard Developer

Guides a new developer through setting up their local environment, understanding the project structure, and making their first contribution.

## Steps

1. [Set Up Environment](1.Set%20Up%20Environment.md) — Install prerequisites and configure local development
2. [Explore the Monorepo](2.Explore%20the%20Monorepo.md) — Understand the directory structure and package relationships
3. [Run the Project](3.Run%20the%20Project.md) — Start dev servers and verify everything works
4. [Understand the Architecture](4.Understand%20the%20Architecture.md) — Learn the data flow from user action to LLM response
5. [Make Your First Change](5.Make%20Your%20First%20Change.md) — Edit code, verify it works, and prepare a contribution

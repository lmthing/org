# Run the Project

Start the local development environment and verify that everything works end-to-end.

## Steps

1. Start all frontend dev servers with `make up`
2. Verify each service is accessible at its `*.test` domain
3. Start Supabase locally with `supabase start`
4. Start edge functions with `supabase functions serve`
5. Test the end-to-end flow: open `studio.test`, check auth redirect, verify agent execution

## Common Issues

- **Port conflicts** — check if another process is using a required port
- **nginx not running** — re-run `make proxy`
- **Supabase not starting** — ensure Docker is running
- **CORS errors** — verify `cloud.test` proxy is configured

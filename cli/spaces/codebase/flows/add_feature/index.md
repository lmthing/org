# Add Feature

Guides a developer through the complete lifecycle of adding a new feature — from planning which packages to touch, through implementation, to shipping.

## Steps

1. [Scope the Feature](1.Scope%20the%20Feature.md) — Identify affected packages and plan the implementation
2. [Set Up the Workspace](2.Set%20Up%20the%20Workspace.md) — Create a branch and prepare the development environment
3. [Implement the Change](3.Implement%20the%20Change.md) — Write code following project conventions
4. [Test and Verify](4.Test%20and%20Verify.md) — Verify the feature works end-to-end
5. [Ship It](5.Ship%20It.md) — Create a PR and prepare for review

# Set Up the Workspace

Prepare the development environment for implementing the feature.

## Steps

1. Create a feature branch from `main`
2. Ensure all dependencies are installed (`pnpm install`)
3. Start the relevant dev servers (only the ones needed for this feature)
4. If the feature involves cloud functions, start Supabase locally
5. Open the key files identified in the scoping step

## Branch Naming

- Feature: `feat/<short-description>`
- Bug fix: `fix/<short-description>`
- Refactor: `refactor/<short-description>`

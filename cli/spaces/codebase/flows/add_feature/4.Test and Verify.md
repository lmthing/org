# Test and Verify

Verify the feature works end-to-end in the local development environment.

## Verification Steps

1. **Visual check** — does the UI look correct? (if frontend change)
2. **Functional check** — does the feature work as intended?
3. **Integration check** — does it interact correctly with other parts of the system?
4. **Regression check** — did anything else break?
5. **Edge cases** — what happens with unexpected input?

## Testing Approaches

- **Frontend**: Open the app in browser, test the feature manually, check console for errors
- **Cloud functions**: Use `curl` or a REST client to test endpoints
- **Shared libs**: Import and test in a consuming app
- **Spaces**: Load the space in Studio and test agent interactions
